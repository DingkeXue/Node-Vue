# Node-Vue
基于 Node Vue 的项目，实现了一个完整的网站基本操作（增删改查）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
